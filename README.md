# 店舗Web運用管理ボード

**デモサイト → https://ssk-t0.github.io/shop-web-operation-demo/**

Web運営スタッフの求人応募用に制作した、管理画面型のポートフォリオです。
店舗運営に関わる**Web更新・出勤管理・広告反響確認・改善アクション管理**を想定し、HTML/CSS/Vanilla JavaScriptのみで実装しています。

---

## 制作目的

「PC操作・Web更新・データ確認ができる」だけでなく、数値を見て次の施策へつなげる運用視点を採用担当者に伝えるために制作しました。
フレームワークに依存せず、基礎的なHTML/CSS/JSの理解をベースにした実装です。

---

## 機能一覧

### 1. 日次確認ボード
- 本日の予約数・問い合わせ数・出勤人数・広告クリック数をカードUIで表示
- スクロール検知でカウントアップアニメーションを実行
- 週間推移グラフ（ミニバーチャート）と直近アクションログを表示
- 数値から見る優先対応・マーケティング視点の改善メモを表示

### 2. 出勤管理テーブル
- スタッフ名・シフト・ステータス・メモを一覧表示
- ステータス（出勤中／休憩中／遅刻／欠勤／退勤）をプルダウンで切り替え可能
- 切り替え時にステータスに応じたカラーバッジが即時反映
- スタッフ名による絞り込み検索と、ステータスフィルターチップ対応

### 3. Web更新フォーム
- タイトル・カテゴリ・本文・公開設定を入力するフォーム
- 入力内容が右側プレビューにリアルタイムで反映される
- 公開 / 下書きの切り替えでプレビューのバッジが変化
- 投稿・下書き保存ボタンでトースト通知を表示
- キャンペーン告知・営業時間・FAQなどの更新優先度を表示

### 4. 広告反響チェック
- 媒体ごとの表示回数・クリック数・問い合わせ数を表示
- CTR（クリック率）・CV率（コンバージョン率）をJavaScriptで自動計算
- CV率はプログレスバーで視覚的に表示
- 全媒体合計のCV率を自動集計してフッターに表示
- 媒体ごとの継続・改善・保留判断と広告運用メモを表示

### 5. バナー管理
- 掲載中バナーのプレビュー・altテキスト・掲載状態を確認
- ホバーで「差し替え」ボタンが表示されるUI
- 掲載中 / 要確認 / 掲載停止をバッジで視認
- CTR・最終更新日など、差し替え判断に必要な補助情報を表示

### 6. 改善メモ
- 集客・コンテンツ・現場運用の3軸で改善案を整理
- 広告数値とWeb更新を分けず、日次運用の次アクションとして見せる構成

### 7. ダークモード / ライトモード切り替え
- ナビゲーションのボタンから表示テーマを切り替え可能
- 選択したテーマをブラウザに保存し、再訪問時も前回の表示を維持
- CSSカスタムプロパティで色を管理し、画面全体の配色を一括で切り替え

## 工夫した点

- **IntersectionObserver** を使ってスクロール時にカウントアップアニメーションを実行し、数値の変化を自然に見せた
- **リアルタイムプレビュー**でWeb更新の操作感をそのまま体験できる仕組みにした
- **ダークモード / ライトモード切り替え**を追加し、ユーザーの閲覧環境に合わせて画面の見やすさを調整できるようにした
- CV率の自動計算は `data-*` 属性から値を取得してJSで処理し、HTMLとロジックを疎結合にした
- 採用担当者が「何の画面か」と「何を判断する画面か」を一目で理解できるよう、数値・判断・次アクションをセットで配置した
- フレームワーク不使用でもレスポンシブ対応・アニメーション・インタラクションを実装し、純粋なJS力を示した

## 想定業務

- スタッフの出勤状況の確認・ステータス管理

- CTRやCV率から媒体ごとの継続・改善・保留を判断
- バナー素材の差し替え依頼・掲載状態の管理
- 現場状況に合わせた予約枠・告知内容・広告導線の調整
